# いきものマスター Ver4.0 開発版

## 今回実装したもの
- 図鑑No.001〜500の固定枠
- 公式図鑑写真と自分の写真を分けて表示する画面
- 写真撮影ごとにカードを1枚追加
- カード所持枚数
- 最初の発見者・初発見日
- 将来の交換機能用 `trades` 保存領域
- AI Worker URLをアプリの設定画面から保存
- 黒・金・緑の新しいPWAアイコン
- バックアップ保存・復元

## 重要
現時点の catalog.js は500枠ありますが、正式な名前・説明が入っているのは68種類です。
残り432種類と、利用許諾を確認した公式写真500枚は未収集です。
写真ファイルは `images/001.webp` 〜 `images/500.webp` を置けば自動表示されます。

## AI接続
Cloudflare Workerを公開後、アプリの「設定」→「AI接続先」に workers.dev URLを入れて保存してください。
worker.js は同梱しています。
