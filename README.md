# いきものマスター Ver3.0 完成版

身近な生きものを撮影し、AI判定・図鑑・カード収集・精霊育成を楽しむWebアプリです。

## 実装済み
- 卵から開始し、1種類で発光、2種類でヒビ、3種類で羽化
- 羽化後の精霊命名と属性判定
- 発見種類数に応じた称号・姿の解放と着せ替え
- カメラ／写真選択／Gemini AIの候補選択
- 生きもの図鑑（写真、説明、レア度、初発見日、場所、保有枚数）
- 重複発見でカード保有枚数を加算
- カード表示は名前・レア度・イラスト・発見者のみ
- お気に入りカード
- フレンド表示（最大10人の準備版）
- プレイヤー名、バックアップ、復元、初期化

## GitHub Pagesへ入れるファイル
`index.html` `config.js` `manifest.webmanifest` `sw.js`

## Cloudflare Worker
`worker.js` をWorkerへ貼り付け、環境変数 `GEMINI_API_KEY` を設定します。公開URLを `config.js` の `AI_WORKER_URL` に入力します。
